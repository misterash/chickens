# Chicken Egg Counter

This is a simple web application written in Python that allows the user to track how many eggs each chicken has laid on each day.

## Features

- Record the number of eggs laid by each chicken on a specific date
- View a summary of eggs laid by each chicken over a specified date range
- Export egg data to CSV format
- Simple and user-friendly web interface

## Requirements
- Python 3.7+
- Flask
